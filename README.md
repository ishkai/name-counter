# Name Counter CLI

Консольная утилита на Go, которая читает файл с именами и считает, сколько раз встречается каждое имя.

Каждое имя должно быть записано на отдельной строке.

## Реализовано:

- чтение файла построчно;
- подсчёт количества повторений каждого имени;
- вывод результата;
- сортировка по имени;
- сортировка по убыванию частоты;
- unit-тесты.

## Установка и запуск

```
git clone https://github.com/ishkai/name-counter.git
cd name-counter
go build -o name-counter
./name-counter <filename.txt>
```


## Использование

```bash
name-counter <filename.txt>
```

Пример файла `names.txt`:

```txt
Алёна
Миша
Алёна
Дима
```

Запуск:

```bash
name-counter names.txt
```

Вывод:

```txt
Алёна:2
Дима:1
Миша:1
```

## Сортировка по частоте

Можно вывести имена в порядке убывания частоты:

```bash
name-counter names.txt --sort
```

Вывод:

```txt
Алёна:2
Дима:1
Миша:1
```

Если частота одинаковая, имена сортируются по алфавиту.
